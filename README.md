# cka-ext-labs
Extra/alternative labs for the CKA course

## Motivation
The aim of these labs is to supplement the official CKx labs with additional, outcome-focused activities which allow learners to practice solving problems which require a combination of concepts to be applied. 

## Labs
* Autoscaling and Performance Management - deploy a webserver, and profile its' response times. Then deploy a HorizontalPodAutoscaler to autoscale the deployment, and observe the impact of this on the performance benchmark.
* NetSec and Troubleshooting - deploy a webserver, exposed by an ingress via cross-namespace routing, whilst adhering to security and resource consumption policies and troubleshooting misconfigurations as necessary.
